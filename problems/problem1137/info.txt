Problem: 1137
Title: N-th Tribonacci Numbers
Category: Fibonacci Style
Link: https://leetcode.com/problems/n-th-tribonacci-number/?envType=study-plan-v2&envId=dynamic-programming
Difficulty: Easy
Question: The Tribonacci sequence T<sub>n</sub> is defined as follows:
T<sub>0</sub> = 0, T<sub>1</sub> = 1, T<sub>2</sub> = 1, and T<sub>n</sub>+3 = T<sub>n</sub> + T<sub>n+1</sub> + T<sub>n+2</sub> for n >= 0.
Given `n`, return the value of T<sub>n</sub>.
